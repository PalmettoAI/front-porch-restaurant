# Front Porch Restaurant & Pie Shop — Website Preview

A concept website preview for **Front Porch Restaurant & Pie Shop**, a family-run Southern home-cooking restaurant and pie shop in Dunnellon, FL (operating since 1986).

- **Live preview:** https://palmettoai.github.io/front-porch-restaurant/
- **Business:** 12039 N Florida Ave, Dunnellon, FL 34434 · (352) 489-4708
- **Google:** 4.4★ from 3,400+ reviews

## About this build

Single-page, dependency-free static site (one `index.html` + optimized JPEGs).

- **Archetype:** hand-painted roadside Americana — cobalt-blue painted-sign motif, chartreuse lettering, weathered cypress textures, butter-cream paper, "pretend it's 1986" manifesto band.
- **SEO:** Restaurant JSON-LD schema, Open Graph + Twitter cards, canonical, sitemap, robots.
- **Accessibility:** semantic landmarks, skip link, alt text, focus-visible, reduced-motion support, accessible hours table.
- **Performance:** lazy-loaded below-fold imagery, preloaded hero, no frameworks, inline critical CSS, font-display swap. Live "open / closed" status computed in the visitor's view from real posted hours (America/New_York).

Photos and review excerpts sourced from the restaurant's public Google listing. This is an unofficial concept preview built by **Palmetto AI Automation** — hours, prices, and menu should be confirmed with the restaurant.
